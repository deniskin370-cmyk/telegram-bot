# 🤖 Telegram Chat Bot

Бот для управления чатами с командами `.spam`, `.mute`, системой ключей и панелью администратора.

## ⚡ Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Настрой переменные окружения

Скопируй `.env.example` в `.env` и заполни:

```bash
cp .env.example .env
```

```env
BOT_TOKEN=токен_от_BotFather
CREATOR_ID=твой_telegram_id
```

> Узнать свой Telegram ID: напиши боту [@userinfobot](https://t.me/userinfobot)

### 3. Запусти бота

```bash
python bot.py
```

---

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `.spam [текст] [кол-во]` | Спамит текст в чат. Максимум **30** сообщений |
| `.mute [время]` | Мутит пользователя (ответь на его сообщение). Форматы: `5m`, `1h`, `2d` |

**Примеры:**
```
.spam Привет! 10
.mute 15m
.mute 2h
.mute 1d
```

---

## 🔑 Система ключей

- Бот работает только у пользователей с активным ключом
- Ключи создаются в **панели администратора**
- Можно задать срок действия ключа: 1 день, 7 дней, 30 дней или навсегда
- Для активации: `/start` → **⚙️ Настройка чатов** → **Ввести ключ**

---

## 👥 Администраторы

- Создатель бота (`CREATOR_ID`) — администратор с самого старта, ключ не нужен
- Добавление/удаление администраторов через панель администратора
- Администраторы могут создавать ключи и управлять другими администраторами

---

## ⚙️ Подключение к чатам (Автоматизация)

1. Открой **Настройки** → **Мой профиль**
2. Нажми **Изменить**
3. Выбери **Автоматизация чатов**
4. Впиши `@wertysrak_bot`

После подключения бот начнёт работать во всех твоих ЛС.
